# Personal

Skills tied to my own setup, not promoted in the plugin.

- **[edit-article](./edit-article/SKILL.md)** — Edit and improve articles by restructuring sections, improving clarity, and tightening prose.
- **[jira-issue-plan](./jira-issue-plan/SKILL.md)** — Read a Jira ticket, grill the user on the implementation plan, then upload the resulting plan as an attachment on the ticket.
- **[jira-plan-to-pr](./jira-plan-to-pr/SKILL.md)** — Take a Jira ticket plus its attached plan, branch off `dev` in every repo touched, deploy to a named testbed, validate, then open PRs into `dev` and post a summary comment on the ticket.
- **[obsidian-vault](./obsidian-vault/SKILL.md)** — Search, create, and manage notes in an Obsidian vault with wikilinks and index notes.
